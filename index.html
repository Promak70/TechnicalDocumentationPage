Introduction_to_HTML

HTML stands for HyperText Markup Language, and it is the standard language used to create web pages.

HTML documents consist of a series of elements that describe the structure and content of the page.

<html>...</html>

CSS_Styling

CSS Styling refers to the way web developers control the look and layout of a webpage by using Cascading Style Sheets (CSS).

It allows for the customization of fonts, colors, spacing, and responsive designs, making websites visually appealing and user-friendly across different devices.

body { background-color: lightblue; }
  • Selectors
  • Box Model
JavaScript_and_Dom

JavaScript is a programming language that enables dynamic interactions on web pages, allowing developers to manipulate content, styles, and attributes.

The DOM (Document Object Model) is an interface that represents the structure of a web document, making it possible for JavaScript to interact with and modify the HTML and CSS of a webpage.

document.querySelector()
  • Event Handling
  • DOM Manipulation
Responsive_Design

Responsive design ensures that web pages adapt smoothly to different screen sizes and devices, providing an optimal viewing experience.

It uses techniques like flexible layouts, media queries, and responsive images to enhance usability across various devices.

@media (max-width: 600px) { }
  • Fluid Layouts
  • Viewport Settings
JavaScript_and_Java

JavaScript and Java are two distinct programming languages, despite sharing similar names. Java is a compiled, object-oriented language commonly used for building large-scale applications,

while JavaScript is a dynamic, interpreted language used primarily for web development and manipulating the Document Object Model (DOM)

console.log("Hello, World!");
  • Execution Environment
  • Object-Oriented Programming
